# Generative UI (GenUI) / A2UI Patterns

Canvas Kit patterns for AI-generated, dynamic user interfaces (GenUI) using the Agent-to-UI (A2UI) protocol. These patterns are designed to render rich components dynamically within conversational interfaces (like Recruiter Hub).

## A2UIRenderer

The `A2UIRenderer` is the core engine that maps JSON payloads into Canvas Kit React components.

**Component**: `design/components/A2UIRenderer.tsx`

**Features**:
- Recursively renders nested `A2UINode` structures.
- Supports native Canvas Kit primitives (`Flex`, `Box`, `Card`, `Heading`, `BodyText`, `PrimaryButton`, `SecondaryButton`, `StatusIndicator`).
- Supports rich, custom GenUI components (`MetricCard`, `JobCard`, `CandidateActionCard`, `DraftMessage`, `CandidateGrid`, `JobReqGrid`, `CandidateCarousel`, `ChartCard`).
- Handles action routing via the `onAction` callback, passing `action` strings and `payload` objects back to the parent component (the agent logic).

**Usage**:
```tsx
import { A2UIRenderer, type A2UINode } from './components';

const node: A2UINode = {
  id: 'example-node',
  component: 'Flex',
  props: { gap: 's', flexDirection: 'column' },
  children: [
    {
      id: 'heading-1',
      component: 'Heading',
      props: { size: 'small', text: 'AI Insight' }
    },
    {
      id: 'btn-1',
      component: 'PrimaryButton',
      props: { text: 'Take Action', action: 'do_something', payload: { id: 123 } }
    }
  ]
};

// Inside a message bubble or dynamic page
<A2UIRenderer node={node} onAction={(action, payload) => handleAction(action, payload)} />
```

## GenUI Components

These are specific, rich components designed to be generated by the AI agent and rendered via the `A2UIRenderer`.

**File**: `design/components/GenUIPatterns.tsx`

### CandidateActionCard
A quick-review card for a candidate, showing their name, metadata, and primary/secondary actions (like Reject/Advance).

**Props**:
- `name` (string)
- `metadata` (string)
- `onReject` (function)
- `onAdvance` (function)

### DraftMessage
A card representing an agent-drafted communication (e.g., an email or Slack message to a hiring manager).

**Props**:
- `recipient` (string)
- `message` (string)
- `onEdit` (function)
- `onSend` (function)

### CandidateGrid & JobReqGrid
Tabular data displays using the Canvas Kit `Table` component. They are designed to show a list of candidates or job requisitions with inline actions.

**Props**:
- `candidates` or `reqs` (array of row objects)
- `onRowAction` (function: `(action: string, payload: any) => void`)

### CandidateCarousel
A horizontally scrolling container for side-by-side comparison of multiple `StructuredResume` components. It forces a minimum width per card to ensure resumes remain readable.

**Props**:
- `candidates` (array of `StructuredResumeProps`)
- `onAction` (function: `(action: string, payload: any) => void`)

### ChartCard
A data visualization wrapper using `react-chartjs-2` inside a Canvas Kit `Card`. Supports line, bar, doughnut, and pie charts.

**Props**:
- `title` (string)
- `type` ('bar' | 'line' | 'doughnut' | 'pie')
- `data` (Chart.js data object)
- `options` (optional Chart.js options object)

## Integration in Conversational Interfaces

When embedding GenUI components inside a chat interface (e.g., using `SanaCommMessageBubble`), ensure the bubble is allowed to expand to the full width of the container so that grids, charts, and carousels have enough room.

**Example**:
```tsx
<SanaCommMessageBubble 
  align="start" 
  maxWidth={msg.a2uiNode ? '100%' : 'min(100%, 600px)'}
  width={msg.a2uiNode ? '100%' : 'auto'}
>
  {msg.text && <Box marginBottom={msg.a2uiNode ? 'm' : 'zero'}>{msg.text}</Box>}
  {msg.a2uiNode && <A2UIRenderer node={msg.a2uiNode} onAction={handleAction} />}
</SanaCommMessageBubble>
```
