## How to Use

### Fetching Confluence Pages
Use Confluence MCP to fetch pages by ID:
```
1. Read this file (confluence-links.md) for URLs
2. Extract pageId from URL
3. Use Confluence MCP to fetch page content
4. Search for relevant strategy context
```

### When to Fetch vs. Use Markdown
- **Use markdown first**: Faster, version-controlled, sufficient for most needs
- **Fetch Confluence when**: Need real-time updates, checking latest decisions, collaborative planning context

### Adding New Links
When you discover a useful strategy Confluence page:
1. Add to this file with clear description
2. Note update frequency and primary use case
3. Keep list curated (remove stale links)
