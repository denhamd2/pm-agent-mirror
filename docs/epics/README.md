# Epic drafts (410)

**410-epic-definition** writes **markdown epic drafts here** — user story, scope, PRD link — **without** creating a Jira epic.

After **420** story-map approval, **430-story-writing** reads the draft from the path recorded in the story map, **creates the Jira epic first**, then creates stories linked to that epic.

Do not expect a Jira epic key until **430** runs.
